# Student Record Management System

## Application
This is a **Python console application** to manage student records efficiently. It allows users to **add, view, search, and delete** student details in a simple and interactive way.

## Main Logic

```
student = {
        "Name": name,
        "Age": age,
        "Course": course,
        "Marks": marks
    }
```

## Features
- **Add Student:** Add student details including ID, name, age, course, and marks.  
- **View Students:** Display details of all students.  
- **Search Student:** Search for a student using their unique ID.  
- **Delete Student:** Remove a student record using their ID.  
- **Exit Program:** Safely exit the application.

## Technologies Used
- **Programming Language:** Python 3  
- **Data Structure:** Dictionary and Set  
- **Interface:** Command-line / Console-based
